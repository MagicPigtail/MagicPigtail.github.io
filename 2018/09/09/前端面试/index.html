css

盒模型

盒模型即把HTML元素看作盒子，在css3中新增属性box-sizing对其设定。
box-sizing有三种值:

content-box模式下:
content宽/高度 = width/height
盒子实际宽/高度 = width/height + padding*2 + border*2
盒子所占宽/高度 = width/height + padding*2 + border*2 + margin*2

border-box模式下:
content宽/高度 = width/height - padding*2 - border*2
盒子实际宽/高度 = width/height
盒子所占宽/高度 = width/height + margin*2

css reset 和 normalize.css 有什么区别？

css reset用于重置浏览器自带样式。
normalize.css是一个可以定制的CSS文件，它让不同的浏览器在渲染网页元素的时候形式更统一。
Normalize.css是一种CSS reset的替代方案。

  1. Normalize.css 保护了有价值的默认值
  2. Normalize.css 修复了浏览器的bug

如何居中？

水平居中
行内:父元素加text-align:center;
块级:margin: 0 auto

垂直居中

  1. 自身高度一半
    position: relative;
    top: 50%;
    margin-top: 自身高度一半;
    缺点:需要知道高度
  2. 借助table-cell
    display:table-cell;
    vertical-align: middle;
    缺点:设置宽高为百分比时失效；容器的margin失效
  3. 借助transform
    position: relative;
    top: 50%;
    transform: translate(0,-50%)
  4. flex布局
    父容器加上
    display: flex;
    align-items: center;
    justify-content: center;
  5. 父容器绝对定位,自身margin: auto

BFC 是什么？

它是页面中的一块渲染区域，有一套渲染规则，决定了其子元素如何布局，以及和其他元素之间的关系和作用。
触发:
满足下列条件之一就可触发BFC:

  1. 根元素，即HTML元素
  2. float的值不为none
  3. overflow的值不为visible
  4. display的值为inline-block、table-cell、table-caption
  5. position的值为absolute或fixed

特性:

  1. 阻止垂直外边距（margin-top、margin-bottom）折叠
  2. 清除内部元素浮动

清除浮动

  1. overflow: hidden
  2. 再父元素中添加clearfix类
